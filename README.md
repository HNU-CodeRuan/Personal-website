## 快速使用

1. fork本项目后再clone到本地修改
2. 进入项目目录执行`npm i`安装依赖
3. 执行`npm run dev`开始开发
4. 修改完后执行`npm run build`发布到ghpages

``` bash
# 安装依赖包
npm install

# 启动程序
npm run dev

# 打包上线
npm run build
```
